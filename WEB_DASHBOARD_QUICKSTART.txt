================================================================================
TROUBLESHOOTING
================================================================================

PROBLEM: Dashboard not loading
SOLUTION: 
  1. Check metrics_api.py is running
  2. Check port 5001 is not in use
  3. Try: http://localhost:5001/api/health

PROBLEM: No data showing
SOLUTION:
  1. Make sure main.py is running with camera
  2. Make sure you've made some hand gestures
  3. Wait 2 seconds for first data point
  4. Click Refresh button

PROBLEM: Charts not updating
SOLUTION:
  1. Check browser console for errors (F12)
  2. Verify API is responding
  3. Restart metrics_api.py

PROBLEM: Memory shows 0 MB
SOLUTION:
  Install psutil: pip install psutil

PROBLEM: Can't connect to localhost:5001
SOLUTION:
  1. Check if metrics_api.py is running
  2. Check if port 5001 is blocked
  3. Try: python metrics_api.py
